# Random Quote Generator

Автор: [Полищук Владимир Артурович]

## Описание

Random Quote Generator - это GUI-приложение на Python, которое позволяет генерировать случайные цитаты из предопределенной коллекции. Приложение сохраняет историю сгенерированных цитат и предоставляет возможности фильтрации по автору и теме.

## Функциональность

- Генерация случайных цитат из коллекции
- Отображение автора и темы цитаты
- Сохранение истории сгенерированных цитат
- Фильтрация цитат по автору и теме
- Добавление новых цитат пользователем
- Сохранение данных в JSON-файлы
- Валидация ввода при добавлении новых цитат
